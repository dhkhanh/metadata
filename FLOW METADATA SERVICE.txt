Luồng metadata-service

API (FormController.java) - api/ui/FormController
 ↓
APPLICATION (FormCommandService, chỉ biết domain.ui.repository.FormRepository)
 ↓
DOMAIN (FormRepository.java) - interface
 ↑
INFRASTRUCTURE ADAPTER (FormRepositoryImpl.java)
 ↑
SPRING DATA JPA (FormDefinitionJpaRepository.java)
 ↑
DATABASE

Giải thích:
- api chỉ nhận HTTP request, gọi Application.
- application điều phối use case.
- domain chứa nghiệp vụ, không biết Spring, JPA, REST.
- infrastructure triển khai các interface mà Domain khai báo.
